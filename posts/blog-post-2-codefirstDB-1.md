# Cómo hacer una base de datos desde cero en 2022 con .NET 6

Si la programación orientada a objetos es lo tuyo, seguramente te has topado con la siguiente incógnita, Cómo hacer para trabajar con una base de datos?

Digamos que los datos son importantes, y necesitamos guardarlos en algun lugar, no? Como cualquier ejemplo que se ocurra: un punto de venta, una tienda en línea, un sistema de administración de stock, etc. Es aquí donde entra la base de datos, la cual necesita almacenar todos estos datos, registrar si hay cambios en ellos, editarlos, modificarlos: crearlos y borrarlos. 
Si tu intuición no te falla, sí , la base de datos solamente es un almacén, como una carpeta en tu explorador. 

Ahora bien, cómo se hace una base de datos? 

Lo primero es saber con qué tecnología y qué proveedor vas a utilizar. 
Existen muchos, como: 
1.  SQLEXPRESS
2.  MONGO DB
3.  MySQL
4.  otro
